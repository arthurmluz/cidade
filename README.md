# Cidade

## compilação (testado apenas em Linux):
`make`
`./main`

## Alterações Disponíveis:
#### GAS_SPAWN       = quantidade de gasolina espalhada pelo mapa
#### GAS_SPAWN_RATE  = chance de spawnar gasolina no mapa
#### TERRAIN_VIEW    = limita a renderização do mapa pra ficar menos pesado
#### BUILDING_RATE   = Chance de um prédio ser spawnado
#### BUILDING_HEIGHT = Altura máxima de um prédio
